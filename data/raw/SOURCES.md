# Sources des données brutes (phase exploration V0)

- Source principale: dépôt GitHub `datasets/s-and-p-500`
  - URL: https://github.com/datasets/s-and-p-500
  - Fichier utilisé: `data/data.csv`
  - URL brute: https://raw.githubusercontent.com/datasets/s-and-p-500/master/data/data.csv
  - Date d'extraction: 2026-03-01

## Variables exploitées dans V0
- `Consumer Price Index` (proxy inflation)
- `SP500` (niveau indice actions)
- `Long Interest Rate` (proxy taux crédit long terme)

## Variables encore non intégrées

- Prix immobilier national

Ces variables nécessitent une source complémentaire et seront ajoutées dans une itération suivante.
